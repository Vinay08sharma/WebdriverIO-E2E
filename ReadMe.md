
# WebDriverIO End to End Framework

Hello, Created an end to end webdriverIO framework.


## To start with

To install all the dependencies

```bash
  npm i
```
    
 To run the spec file

```bash
  npm run test
```

To open allure report

```bash
  npm run allure-report
```


## Tech Stack

Typescript, Mocha
